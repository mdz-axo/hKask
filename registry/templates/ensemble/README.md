# Ensemble Templates — Deferred (2026-06-14)

The ensemble module has been removed from hKask. These templates are preserved
as reference for a future multi-agent mode that will evolve from the dual-presence
pattern (see `docs/specifications/dual-presence-pattern.md`).

**Reactivation criterion:** When dual-presence has produced N≥3 stable sessions
with distinct ACP agents.

## Preserved Templates

| Template | Purpose |
|----------|---------|
| `standing_session_administrator_view.j2` | Administrator view of standing session |
| `standing_session_curator_instruction.j2` | Curator instruction for standing session |
| `standing_session_metacognition_update.j2` | Metacognition update template |
| `standing_session_status_report.j2` | Status report template |
| `selectors/participant-selector.j2` | Participant selection logic |
| `selectors/relevance-check.j2` | Relevance self-assessment |
