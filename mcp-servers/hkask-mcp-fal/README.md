# hkask-mcp-fal

Fal workflow execution MCP server — exposes `fal.execute_workflow` tool for Strategy D PDCA loops.

## Tools

| Tool | Description |
|------|-------------|
| `fal.execute_workflow` | Execute a fal.ai workflow with the given input parameters |

## Environment Variables

| Variable | Required | Description |
|----------|----------|-------------|
| `FA_API_KEY` | Yes | fal.ai API key for workflow execution |
