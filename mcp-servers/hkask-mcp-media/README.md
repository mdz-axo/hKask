# hkask-mcp-media

Media generation MCP server — image, video, audio, and 3D generation via fal.ai and other providers.

## Tools (15)

| Tool | Description |
|------|-------------|
| `detect_faces` | Detect faces in image |
| `match_faces` | Match faces between images |
| `validate_face_reference` | Validate face reference image |
| `analyze_colors` | Analyze image colors |
| `analyze_composition` | Analyze image composition |
| `add_caption` | Add caption to image |
| `caption_scene` | Caption a scene |
| `detect_objects` | Detect objects in image |
| `capture_audio` | Capture audio |
| `extract_keyframes` | Extract keyframes from video |
| `images_to_video` | Convert images to video |
| `clip` | Clip video segment |
| `concat` | Concatenate videos |
| `to_gif` | Convert video to GIF |
| `run` | Main run loop |

## Configuration

Provider API keys configured via standard environment variables (`FW_API_KEY`, `FAL_KEY`, etc.).
