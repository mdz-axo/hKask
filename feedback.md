# hKask Feedback Ledger

User-submitted onboarding and usability notes.
Collected via `/feedback` in the interactive REPL.

Each entry format:

```
## YYYY-MM-DD HH:MM:SS UTC — <replicant>

> <comment>

---
```

**Location at runtime:** `~/.local/share/hkask/feedback.md`

This file in the repository is the schema reference only.
Runtime entries are written to the user's local data directory, not here.
