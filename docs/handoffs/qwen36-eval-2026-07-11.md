## 5. Key Decisions to Preserve

1. **Adapters trained with Unsloth MUST be loaded with Unsloth.** Unsloth's `FastLanguageModel` wraps modules with names like `model.language_model.layers.*`, and the `adapter_config.json` target_modules regex requires this prefix. Vanilla `transformers` module names (`model.layers.*`) don't match, causing silent 0-parameter injection.
2. **`temperature=0.0` is invalid for Unsloth's generate.** Pass `temperature=1.0, do_sample=False` for deterministic output.
3. **Qwen3.6 outputs thinking traces.** The model was trained on reasoning data and wraps answers in `<think>...</think>\n\n[answer]`. Eval code must extract the letter/number after `</think>`, not just take `response[0]`.
4. **Container disk is only 60GB; model cache must live on `/workspace`.** The 200GB `/workspace` network volume is the right place for HF cache. Current setup uses a symlink from `/root/.cache/huggingface/hub` → `/workspace/.cache/huggingface/hub`.
5. **Do not revert to `resolve/main` HF URLs.** Immutable commit-pinned URLs prevent CDN staleness.
6. **GPQA is gated; requires `HF_TOKEN`.** Recover from `/proc/1/environ` on the pod.
