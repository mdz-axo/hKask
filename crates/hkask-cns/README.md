# hkask-cns

Cybernetic Nervous System for hKask.

System-wide observability: span emission, variety counters, algedonic alerts.

## Key Concepts

| Concept | Description |
|---------|-------------|
| **CNS spans** | `cns.*` structured observability events |
| **Variety counters** | Ashby's Law — measure environmental complexity |
| **Algedonic alerts** | Pain/pleasure signals — escalate to Curator on deficit |
| **Set points** | Configurable thresholds for alert triggers |
| **Energy budget** | Token/cost tracking per operation |

## Span Registry

Canonical span definitions in `hkask-types::cns::CnsSpan` (84+ spans).
