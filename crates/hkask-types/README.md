# hkask-types

ID types, nu-event, and visibility types for hKask.

Foundation crate — all other crates depend on this. Defines the core domain vocabulary.

## Key Types

| Module | Purpose |
|--------|---------|
| `cns` | `CnsSpan` registry — canonical CNS span definitions |
| `id` | `WebID`, `AgentID`, `PodID`, other identity types |
| `nu_event` | ν-event types for CNS observability |
| `visibility` | Visibility and access control types |
| `sovereignty` | Sovereignty enforcement types |
| `kanban` | Kanban task and board types |
| `loops` | Cybernetic loop channel types |

**LOC:** ~12,500 (largest crate)
