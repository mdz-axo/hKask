# hkask-improv

Improv interaction protocol for hKask agents.

Constructive-by-default communication protocol for dual-presence chat (human + agent).

## Modes

| Mode | Description |
|------|-------------|
| **Plussing** | Build on the previous contribution — "yes, and also..." |
| **Yes And** | Accept and extend — full affirmation with extension |
| **Yes But** | Accept with constraint — affirmation with bounded objection |
| **Freestyling** | Open-ended creative exploration |
| **Riffing** | Thematic variation on an established pattern |

Activate in REPL with `/improv <mode>`.
