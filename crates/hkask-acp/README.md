# hkask-acp

ACP (Agent Client Protocol) userpod for hKask.

Presents hKask coding agents in IDEs via the Agent Client Protocol — enables `kask` to operate as a Zed coding agent.

## Features

- **IDE integration** — presents hKask agents as IDE tool-using agents
- **UserPod mode** — agent runs as a userpod with authorial style
- **Tool passthrough** — IDE tools (files, terminal, search) available to hKask agents
